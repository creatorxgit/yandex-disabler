# Yandex Disabler
# Блокировщик Яндекс Браузера для Windows 10 и 11
# Как установить?
## 1. Устанавливаем дополнительные программы
### Заходим в комадную строку и вводим pip install python
## 2. Скачиваем фаил
### Выбераем последнию версию блокировщика
### Скачиваем yandex_disabler.py
### Ждем когда скачается фаил.
## 3. Открываем его
### Если появляется диалоговое окно с просьбой выбрать приложение, выберите Python 
## 4. Настроиваем программу 
### Сначала у нас появится окно которе требует открыть от имени администратора. Выбераем Да
### После этого как вы нажали на кнопку Да мы увидем интерфейс программы. Нажимаем на кнопку Заблокировать.
### Закрываем окно.
## Готово! Тепреь у нас заблокирован Яндекс Браузер!
